## Reproducing these examples locally

After installing dependencies and running `scripts/merge_datasets.py` + `scripts/build_index.py`:

```bash
# Example 1 — experimental design
python -m tools.filter classify-query "CO2 reduction in water microdroplets" --taxonomy reaction
python -m tools.search "CO2 reduction water microdroplets" --k 12
python -m tools.search "..." --k 20 | python -m tools.combine --top-n 5
python -m tools.cite Deepseek_00498 Qwen_00131 ... --style numbered

# Example 2 — Gaussian cluster QC
python -m tools.build_model cluster --solute-smiles "OO" --n-waters 20 -o models/h2o2_cluster.xyz
python -m tools.write_input --xyz models/h2o2_cluster.xyz --code gaussian \
    --method wB97X-D --basis "6-31+G(d,p)" -o inputs/h2o2_opt.gjf

# Example 3 — CP2K AIMD slab
python -m tools.build_model slab --solute-smiles "O=C=O" \
    --n-waters-per-layer 24 --n-layers 4 -o models/co2_slab.xyz
python -m tools.write_input --xyz models/co2_slab.xyz --code cp2k \
    --cell models/co2_slab.cell --xc BLYP --steps 10000 -o inputs/co2_aimd.inp
```

Or just open Claude Code in the project directory and ask the same questions
in natural language — the skill and subagent will do this for you.

或者直接在项目根目录开 Claude Code 用自然语言提问 — skill 和子代理会替你完成
上述命令调用。
